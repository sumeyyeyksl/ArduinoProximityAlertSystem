# Arduino Proximity Alert System

A simple Arduino Uno project that measures distance using an ultrasonic sensor and provides visual and audio alerts.

## Features

- Distance measurement with HC-SR04 ultrasonic sensor
- Green, yellow, and red LED indicators
- Buzzer warning based on object distance

## Technologies

- C++
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
